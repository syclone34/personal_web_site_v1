# Personal Portfolio Website

A modern, responsive portfolio website for web developers.

## Features

- **Responsive Design**: Works seamlessly on desktop, tablet, and mobile devices
- **Modern UI**: Clean and professional design with smooth animations
- **Sections Include**:
  - Hero section with call-to-action buttons
  - About section with personal information
  - Skills showcase with technology icons
  - Featured projects with links to demos and code
  - Contact form and social media links
  - Fixed navigation with active state tracking
  - Scroll-to-top button

## Technologies Used

- HTML5
- CSS3 (with CSS Grid and Flexbox)
- Vanilla JavaScript
- Font Awesome icons

## Setup Instructions

1. Clone or download this repository
2. Replace placeholder content with your personal information:
   - Update "Your Name" throughout the files
   - Add your contact information in the About section
   - Update social media links in the Contact section
   - Replace project information with your actual projects
3. Add your project images to `assets/images/` folder:
   - project1.jpg
   - project2.jpg
   - project3.jpg
4. Open `index.html` in your browser to view the website

## Customization

### Colors
Edit the CSS variables in `styles.css` to change the color scheme:
```css
:root {
    --primary-color: #3b82f6;
    --secondary-color: #1e40af;
    --accent-color: #60a5fa;
    /* ... more variables */
}
```

### Adding More Projects
Add additional project cards in the Projects section of `index.html`:
```html
<div class="project-card">
    <!-- Project content here -->
</div>
```

### Skills
Update the skills section by adding or removing skill cards in `index.html`.
