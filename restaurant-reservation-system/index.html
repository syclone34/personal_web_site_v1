Restaurant Reservation System

A full-stack booking platform with React, Node.js, and SQLite

React Node.js Express SQLite

Key Features

  • Real-time table availability checking
  • Customer reservation management
  • Admin dashboard for restaurant staff
  • Automatic table assignment by party size
  • SQLite database with auto-initialization
Note: This is a full-stack application that requires both frontend and backend servers to be running. It cannot be opened directly as a static HTML file.

Quick Start

To run this application locally:

# Terminal 1 - Start Backend Server cd restaurant-reservation-system/server npm install npm start # Terminal 2 - Start Frontend App cd restaurant-reservation-system/client npm install npm start # App opens at http://localhost:3000

Deployment

This application is ready to deploy to:

  • Frontend: Vercel, Netlify, or GitHub Pages
  • Backend: Render, Railway, or Heroku

See QUICK_START_DEPLOY.md for detailed deployment instructions.
